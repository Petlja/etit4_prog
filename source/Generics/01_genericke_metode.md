# Генеричке методе

Генеричке методе у програмском језику C# омогућавају писање метода које могу да
раде са различитим типовима података. Користе типске параметре, обично означене
словом `T`, чиме се омогућује да метода буде независна од конкретног типа
података. Дефинишу се додавањем типског параметра `<T>` између имена методе и
њених параметара. Синтакса генеричке методе која има један параметар изгледа
овако:

```cs
public static void GenerickaMetoda<T>(T parametar)
{
    // Telo metode
}
```

Једноставан пример била би метода за испис вредности било ког конкретног типа:

```cs
public static void Ispisi<T>(T parametar)
{
    Console.WriteLine(parametar);
}
```

Ова метода прихвата типски параметар `T` који се одређује у тренутку њеног
позива, тј. параметар `T vrednost` означава да метода прихвата променљиву било
ког типа `T` (нпр. `int`, `double` или `string`). Захваљујући типском параметру
`T`, методу `Ispisi` можеш да позовеш са аргументом било ког конкретног типа:

```cs
static void Main()
{
    int ceoBroj = 1;
    double realanBroj = 2.2;
    string niska = "tri";
    Ispisi<int>(ceoBroj);
    Ispisi<double>(realanBroj);
    Ispisi<string>(niska);
}
```

Типски параметар `<T>` наведен у дефиницији методе није потребно наводити
приликом позива – компајлер ће сâм одредити тип на основу прослеђеног
аргумента:

```cs
static void Main()
{
    int ceoBroj = 1;
    double realanBroj = 2.2;
    string niska = "tri";
    Ispisi(ceoBroj);
    Ispisi(realanBroj);
    Ispisi(niska);
}
```

Аргумент не мора бити променљива – може бити наведен и као вредност. Тако,
претходни програм може бити написан и овако:

```cs
static void Main()
{
    Ispisi(1);
    Ispisi(2.2);
    Ispisi("tri");
}
```

Након извршења овог програма у конзоли ће се исписати:

```text
1
2.2
tri
```

Нека је задатак да напишеш методу `IspisiTip` која ће се користити за проверу и
испис типа података било које вредности или променљиве која се приликом позива
наведе као њен аргумент. Решење задатка може да изгледа овако:

```cs
public static void IspisiTip<T>(T parametar)
{
    Console.WriteLine("{0} je tipa {1}", parametar, typeof(T));
}
```

Оператор `typeof` користи се за добијање типа податка у току извршавања
програма, а у овом случају, `typeof(T)` враћа тип податка који је прослеђен као
аргумент. Ако се метода `IspisiTip` позове са аргументом било ког конкретног
типа, на пример...

```cs
static void Main()
{
    IspisiTip(true);
    IspisiTip(1);
    IspisiTip(2.2);
    IspisiTip("tri");
}
```

...у конзоли ће се исписати:

```text
True je tipa System.Boolean
1 je tipa System.Int32
2.2 je tipa System.Double
tri je tipa System.String
```

Мало сложенији пример била би метода за испис елемената низа било ког
конкретног типа. Тако, у методи `IspisiNiz`...

```cs
public static void IspisiNiz<T>(T[] niz)
{
    foreach (T element in niz)
    {
        Console.WriteLine(element);
    }
}
```

...аргумент типа `<T>` означава да метода прихвата типски параметар `T` који се
одређује у тренутку позива, тј. `T[] niz` означава да метода прихвата низ било
ког типа `T[]` (нпр. `int[]`, `double[]` или `string[]`). Метода `IspisiNiz`
сада може да прихвати низ било ког конкретног типа:

```cs
static void Main()
{
    int[] celiBrojevi = { 1, 2, 3 };
    double[] realniBrojevi = { 1.2, 3.4, 5.6 };
    string[] niske = { "jedan", "dva", "tri" };
    IspisiNiz(celiBrojevi);
    IspisiNiz(realniBrojevi);
    IspisiNiz(niske);
}
```

Након извршења овог програма у конзоли ће се исписати:

```text
1
2
3
1.2
3.4
5.6
jedan
dva
tri
```

Као што је приказано у примерима, генеричке методе омогућавају писање општег и
поново употребљивог кода који ради са различитим типовима. Оне поједностављују
имплементацију, смањују могућност грешака и чине кôд лакшим за читање и
одржавање.

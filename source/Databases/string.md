# Конекциони стринг

Конекциони стринг је стринг који садржи све информације неопходне апликацији
за повезивање на базу података. Конекциони стринг укључује:

* име или IP адреса хоста на којем се налази SQL Server,
* име базе података,
* корисничко име и лозинку (ако су потребни), као и
* додатне опције попут времена тајмаута, типа енкрипције итд.

## Добијање конекционог стринга

У интегрисаном развојном окружењу *Visual Studio*, у менију *View* одабери
**Server Explorer**.

Десни кликом на **Data Connections** одабери **Add Connection**. У
*Add Connection* прозору одабери тип базе (Microsoft SQL Server), унеси име
сервера, одабери базу података и начин аутентификације.
